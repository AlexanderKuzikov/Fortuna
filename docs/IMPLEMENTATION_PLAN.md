## Риски и ограничения

1. **Отсутствие исходного кода**: Сайт — статический билд. Изменения вносятся напрямую в HTML. Это усложняет поддержку и требует осторожности.
2. **Нет фотографий товаров**: Критический блокер для фазы 2. Нужны фото от заказчика.
3. **Нет JavaScript**: Невозможна реализация интерактивных элементов без добавления JS в билд.
4. **NPM/сборка**: Если нет доступа к исходникам TanStack Start, все изменения ручные.
